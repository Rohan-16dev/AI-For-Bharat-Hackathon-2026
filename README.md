🚀 UBID Intelligence Platform

An AI-powered industrial intelligence system that analyzes entity data, detects inconsistencies, and provides strategic insights using Google Gemini.

---

🧠 Features

- 🔍 Deep entity analysis (PAN, GSTIN, activity signals)
- 🔗 Hidden linkage detection across datasets
- 📊 Operational status prediction (Active / Dormant / Closed)
- 🤖 AI-powered chatbot for interactive insights
- 🧾 Automated anomaly detection & resolution suggestions

---

⚙️ Tech Stack

- React + TypeScript (Frontend)
- Vite (Build Tool)
- TailwindCSS (Styling)
- Google Gemini API (AI Engine)

---

🔐 Environment Setup

Create a ".env.local" file:

VITE_GEMINI_API_KEY=your_api_key_here

---

▶️ Run Locally

npm install
npm run dev

---

📌 Notes

- This project uses the Gemini API (free tier supported)
- No backend required — runs fully client-side
